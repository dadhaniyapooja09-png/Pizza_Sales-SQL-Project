 **🍕 Pizza Sales Analysis Using SQL**

## 📌 Project Overview

This project focuses on analyzing pizza sales data using SQL to uncover meaningful business insights related to customer ordering behavior, revenue generation, sales trends, and product performance. The analysis was performed using advanced SQL queries, joins, aggregate functions, subqueries, and window functions.

The project demonstrates how raw business data can be transformed into actionable insights that support better decision-making, inventory planning, and sales optimization.

## 🎯 Objectives

* Analyze overall pizza sales performance
* Identify top-selling pizza types and categories
* Calculate total revenue generated
* Discover customer ordering patterns
* Determine peak sales hours and popular pizza sizes
* Analyze category-wise sales contribution
* Generate business insights using SQL queries

📂 Dataset Information

# Tables Used
  **Orders** – Contains order date and time details
  **Order Details** – Stores quantity and pizza-wise order records
  **Pizzas** – Includes pizza sizes and pricing information
  **Pizza Types** – Contains pizza names, categories, and ingredients
